== Installation == 

1. Upload the `gpt-ai-content-creator` folder to `/wp-content/plugins/` directory.
2. Activate the plugin using the 'Plugins' menu in your WordPress admin panel.
3. You can adjust the necessary settings using your WordPress admin panel in "GPT AI Content Creator".

== Frequently Asked Questions == 

= What is the purpose of the GPT Chat WordPress plugin? =

The GPT Chat WordPress plugin is a tool designed to generate content for blog posts using GPT (Generative Pretrained Transformer) technology. It allows users to create posts based on a title by calling the OpenAI GPT API.

= Do I need any technical skills to use this plugin? =

No, you don't need any technical skills to use the GPT Chat WordPress plugin. It's designed to be user-friendly and easy to use, even if you have no coding experience.

= How do I configure the API key, max tokens, and temperature settings? =

To configure these settings, you'll need to go to the plugin settings page in the WordPress dashboard. There, you can enter your API key and adjust the max tokens and temperature settings to suit your needs.

= What happens if the GPT API fails to generate content? =

If the GPT API fails to generate content, the plugin will return an error message to the user. This could happen if the API key is invalid or if there's a problem with the API itself.

= I have some problems with the plugin's work. What Information should I provide to receive proper support? =

Please make sure that the problem hasn't been discussed yet on our forum (<https://support.bestwebsoft.com>). If no, please provide the following data along with your problem's description:
